# Samaya

### Samaya is a clock project having functionalities such as showing default time, activating Timer and the Stopwatch.

To use this project, you need to add following dependencies to the file 'pubspec.yaml'.
- numberpicker: ^1.0.0
- google_fonts: ^1.1.0
